About the System

I am quite a disorganised person by nature but I want to change this so I have created a system. I've stolen, modified, invented, played around and thrown away things that I have learned from others so I'm writing up my current system. I hope by writing this down I can tweak, modify, rip out and inspire myself to improve and think critically about my system, and maybe help others too.
